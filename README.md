# Building RP Bots for Internet-Connected LLMs

## 🇬🇧 English

Documentation and practical workflows for building roleplay bots for internet-connected LLMs.

This repository focuses on long-term behavioral stability, practical script architecture, and design methods for creating roleplay bots that remain engaging over extended interactions.

## 🇫🇷 Français

Documentation et méthodes de travail pour créer des bots de roleplay destinés à des LLM connectés à Internet.

Ce dépôt met l'accent sur la stabilité comportementale à long terme, l'architecture pratique des scripts et les méthodes de conception permettant de créer des bots agréables à jouer sur la durée.

---

# 🧭 New to the Repository?

If this is your first time reading this repository, I recommend following this reading order.

The shared guides introduce the core design principles used throughout every workflow.

After that, choose either the canon or original workflow depending on the type of bot you want to build.

Finally, use the production templates to turn the workflow into a complete script.

**Recommended reading order**

1. How to Build Participating Characters
2. How to Protect Player Agency in RP Bots
3. How to Build Anti-Loop Systems for RP Bots
4. How to Build Specialized Modules for RP Bots
5. How to Test and Debug RP Bots
6. Canon or Original Workflow
7. Introduction and Starter Workflow
8. Production Templates

# 📁 Repository Structure

The repository is organized into four main sections.

- `shared/` — Design principles that apply to every RP bot.
- `canon/` — Workflows dedicated to canon characters.
- `original/` — Workflows dedicated to original characters.
- `templates/` — Production templates for building complete bot scripts.

Each section is available in both English and French.

# ℹ️ About This Repository

## 🇬🇧 English

This repository gathers documentation for building roleplay bots for internet-connected LLMs, with a focus on:

- long-term behavioral stability
- script architecture
- character participation
- player agency
- anti-loop systems
- specialized module design
- testing and debugging methods
- introductions and starters
- production-ready script templates
- practical workflows for RP platforms

The goal is not to provide universal rules for every roleplay bot.

Instead, this repository documents the workflow and design principles that currently produce the most consistent results throughout my own projects.

---

## 🇫🇷 Français

Ce dépôt regroupe de la documentation pour créer des bots de roleplay destinés à des LLM connectés à Internet, avec un accent mis sur :

- la stabilité comportementale à long terme
- l'architecture des scripts
- les personnages participatifs
- la protection du joueur
- les systèmes anti-loop
- la conception de modules spécialisés
- les méthodes de test et de debugging
- les introductions et démarrages
- les templates de scripts prêts à l'emploi
- les méthodes de travail pratiques pour les plateformes de roleplay

L'objectif n'est pas de proposer une méthode universelle applicable à tous les bots de roleplay.

Ce dépôt documente simplement le workflow et les principes de conception qui produisent actuellement les résultats les plus stables dans mes propres projets.
